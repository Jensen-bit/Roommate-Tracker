# Roommate-Tracker
A web application built to streamline shared living for college students.

## Weekly Questions

- What did you do
- challenges during implementations
- what are you planning for next week

## Features
- Shared Chore Tracking
- Real-time Grocery Lists
- Household Announcements
- Integrated Expense Management

## Tech Stack
- **Frontend:** HTML/CSS/JS
- **Backend:** Node.js, Express, Nodemailer
- **Database:** PostgreSQL
- **Environment:** Docker
